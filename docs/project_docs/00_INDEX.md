# Monopoly SDS Documentation Pack

This folder contains all planning and product docs required before implementation start.

## Files
- `01_BRAND_UX_SDS.md` - SDS-inspired visual direction and UX baseline.
- `02_MULTIPLAYER_RULES.md` - locked multiplayer and economy rules.
- `03_PRD_MVP_MLP.md` - product requirements and MVP acceptance.
- `04_SYSTEM_ARCHITECTURE.md` - technical architecture and core flows.
- `05_DELIVERY_ROADMAP.md` - phased delivery plan.
- `06_AI_ASSET_PROMPTS.md` - AI prompts for all required visual assets.
- `07_OPEN_QUESTIONS.md` - unresolved decisions to close before coding.

## Note
Brand direction is based on reference site: `https://www.sds-group.ru/`.
Final brand colors and logo usage should be validated against official SDS brand guidelines if available.
